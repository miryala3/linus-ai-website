Installation

Installation Instructions

Get up and running in under 60 seconds on any platform.

macOS — zsh
$ cd ~/Downloads
$ chmod +x linus-ai-macos-arm64
$ ./linus-ai-macos-arm64 --version
LINUS-AI v2.0.1 (darwin-arm64)
 
# Move to PATH for global access:
$ sudo mv linus-ai-macos-arm64 /usr/local/bin/linus-ai
$ linus-ai --chat "Hello, Linus!"
 
# Or use the DMG installer (drag & drop):
$ open linus-ai-macos.dmg
Gatekeeper Notice: On first launch macOS may show a security warning. Right-click the binary → Open → click "Open". Alternatively: xattr -dr com.apple.quarantine linus-ai-macos-arm64
Linux — bash (bare binary)
$ chmod +x linus-ai-linux-x86_64
$ ./linus-ai-linux-x86_64 --version
LINUS-AI v2.0.1 (linux-x86_64)
 
# Install system-wide:
$ sudo install -m 755 linus-ai-linux-x86_64 /usr/local/bin/linus-ai
$ linus-ai --activate LNAI-XXXX-XXXX-XXXX-XXXX
$ linus-ai --serve # Start API server on :8080
Debian / Ubuntu / Mint — bash
# Install via dpkg (replaces prior version automatically):
$ sudo dpkg -i linus-ai_2.0.1_amd64.deb
Setting up linus-ai (2.0.1) ...
 
$ linus-ai --version
LINUS-AI v2.0.1 (linux-x86_64)
 
$ linus-ai --activate LNAI-XXXX-XXXX-XXXX-XXXX
$ linus-ai --serve
 
# Uninstall:
$ sudo dpkg -r linus-ai
RHEL / Fedora / CentOS / openSUSE — bash
# Fedora / RHEL 8+:
$ sudo rpm -i linus-ai-2.0.1-1.x86_64.rpm
 
# Or with dnf (handles deps):
$ sudo dnf install ./linus-ai-2.0.1-1.x86_64.rpm
Installed: linus-ai-2.0.1-1.x86_64
 
$ linus-ai --activate LNAI-XXXX-XXXX-XXXX-XXXX
$ linus-ai --serve
 
# Uninstall:
$ sudo rpm -e linus-ai
Windows — PowerShell
# Option A: MSI installer (recommended — adds to PATH automatically):
Double-click linus-ai-windows-x86_64.msi → Next → Install → Finish
 
# Option B: bare .exe via PowerShell:
PS> .\linus-ai-windows-x86_64.exe --version
LINUS-AI v2.0.1 (windows-x86_64)
 
# Add to PATH manually (run as Administrator):
PS> Move-Item .\linus-ai-windows-x86_64.exe C:\Windows\System32\linus-ai.exe
 
PS> linus-ai --activate LNAI-XXXX-XXXX-XXXX-XXXX
PS> linus-ai --chat "Hello!"
Windows Defender: SmartScreen may flag the binary on first run. Click "More info" → "Run anyway". Verify the SHA256 checksum shown on each card above.
Security

Verify Your Download

Always verify the SHA256 checksum of downloaded binaries. The complete SHA256SUMS.txt is attached to every release.

Checksum verification
# Linux / macOS — verify a single file:
$ sha256sum linus-ai-linux-x86_64
 
# macOS (shasum):
$ shasum -a 256 linus-ai-macos-arm64
 
# Batch verify all downloaded files against SHA256SUMS.txt:
$ sha256sum --check SHA256SUMS.txt
linus-ai-linux-x86_64: OK
linus-ai-macos-arm64: OK
...
 
# Windows PowerShell:
PS> Get-FileHash .\linus-ai-windows-x86_64.msi -Algorithm SHA256 | Select Hash

SHA256SUMS.txt is attached to every release at github.com/LINUS-AI-PRO/linus-ai/releases. Individual checksums are also shown on each download card above (click "Show SHA256").

Release Highlights

What's new in LINUS-AI 2.0.1.

📦 Native Installers — DMG, MSI, .deb, .rpm New

Install LINUS-AI like any native app. macOS DMG (drag & drop), Windows MSI (adds to PATH automatically), Debian/Ubuntu .deb (dpkg/apt), and RHEL/Fedora .rpm (rpm/dnf) packages — all architectures, all distros.

📱 Android & iOS Builds New

Official native binaries for Android arm64 (API 26+) and iOS arm64 / Simulator. Embed the LINUS-AI inference engine directly in your mobile app.

⚡ Tensor Parallelism Across Multiple GPUs v1.4

Split large models across up to 8 GPUs with automatic tensor sharding. 70B parameter models run in real-time on 4× RTX 3090s.

🕸 Mesh Networking Mode v1.4

Distribute inference across multiple machines on your private network. Nodes auto-discover each other using mDNS. No Kubernetes required.

🔐 Vault — Encrypted Memory Layer v1.4

All conversation history and embeddings stored AES-256-GCM encrypted at rest. The vault key never leaves your machine.
